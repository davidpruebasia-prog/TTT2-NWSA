# Checklist de Reunión Diaria de Seguridad - NWSA TTT-2

## ✅ Lista de Verificación para Reuniones Efectivas

### PREPARACIÓN PRE-REUNIÓN (15 minutos antes)

#### Documentación Requerida
- [ ] Formato de reunión diaria disponible
- [ ] Lista de asistencia preparada
- [ ] Reporte del día anterior revisado
- [ ] Plan del día actual actualizado
- [ ] Evaluación de riesgos del sitio completada

#### Información para Compartir
- [ ] Condiciones climáticas verificadas
- [ ] Incidentes/eventos del día anterior documentados
- [ ] Tareas planificadas para el día identificadas
- [ ] Riesgos específicos evaluados
- [ ] Controles requeridos determinados

#### Logística
- [ ] Área de reunión preparada y segura
- [ ] Equipos de demostración disponibles (si aplica)
- [ ] Material de referencia accesible
- [ ] Tiempo asignado respetado

### DURANTE LA REUNIÓN (Máximo 15 minutos)

#### Apertura (2 minutos)
- [ ] Saludo y bienvenida al personal
- [ ] Verificación de asistencia completa
- [ ] Confirmación de que todos pueden escuchar/ver
- [ ] Establecimiento del propósito de la reunión

#### Revisión del Día Anterior (3 minutos)
- [ ] Incidentes o casi accidentes reportados
- [ ] Lecciones aprendidas compartidas
- [ ] Reconocimientos positivos dados
- [ ] Acciones correctivas revisadas

#### Plan del Día Actual (5 minutos)
- [ ] Tareas principales comunicadas claramente
- [ ] Asignaciones de personal especificadas
- [ ] Riesgos específicos identificados y discutidos
- [ ] Controles y EPP requeridos explicados

#### Procedimientos Críticos (3 minutos)
- [ ] Procedimientos de trabajo seguro revisados
- [ ] Comunicaciones de emergencia recordadas
- [ ] Puntos de reunión y rutas de evacuación confirmadas
- [ ] Sistema de comunicación verificado

#### Verificación de Comprensión (2 minutos)
- [ ] Preguntas de verificación realizadas
- [ ] Participación activa del personal fomentada
- [ ] Dudas o preocupaciones abordadas
- [ ] Demostraciones prácticas realizadas (si necesarias)

### CIERRE DE REUNIÓN

#### Compromisos y Seguimiento
- [ ] Acuerdos específicos establecidos
- [ ] Responsables designados claramente
- [ ] Fechas límite definidas
- [ ] Próxima reunión programada

#### Documentación
- [ ] Lista de asistencia firmada por todos
- [ ] Minuta de reunión completada
- [ ] Acciones de seguimiento documentadas
- [ ] Observaciones registradas

### POST-REUNIÓN

#### Distribución y Archivo
- [ ] Minuta distribuida a participantes
- [ ] Copia enviada a supervisión
- [ ] Documentación archivada correctamente
- [ ] Seguimiento de acciones iniciado

#### Evaluación de Efectividad
- [ ] Participación del personal evaluada
- [ ] Comprensión de contenidos verificada
- [ ] Tiempo de reunión apropiado
- [ ] Mejoras identificadas para próximas reuniones

### INDICADORES DE CALIDAD

#### Participación
- **Óptimo:** 100% del personal participa activamente
- **Aceptable:** 90% participación con engagement
- **Requiere mejora:** <90% participación o participación pasiva

#### Comprensión
- **Óptimo:** Todo el personal demuestra comprensión completa
- **Aceptable:** La mayoría comprende, pocas dudas
- **Requiere mejora:** Dudas significativas o mala comprensión

#### Documentación
- **Óptimo:** Completa, precisa y archivada inmediatamente
- **Aceptable:** Completa con pequeños retrasos en archivo
- **Requiere mejora:** Incompleta o archivo deficiente
